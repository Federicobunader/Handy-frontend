Alquilá tus herramientas

con Handy

Handy es una página web orientada al leasing de herramientas en la cual los usuarios pueden tanto subir publicaciones de herramientas para poner el alquiler, como buscar herramientas para alquilar y hacer uso de ellas. Todos estos alquileres pueden ser con posibilidad de compra, siempre y cuando quien alquila lo decida.
